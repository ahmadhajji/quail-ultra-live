# Quail Ultra 0.9.0 Late Beta 1

Quail Ultra is a late-beta desktop question-bank browser built from Quail and reshaped around a more UWorld-like review workflow.

## Highlights

- explicit `Tutor`, `Timed`, and `Untimed` block modes
- continuous question -> answer -> explanation solving flow
- per-question answer elimination and multicolor highlighting
- normalized legacy progress loading for older Quail sessions
- simplified exam UI and updated builder, overview, and previous-block screens
- separate app identity from original Quail, so both apps can coexist on the same machine

## Platform Notes

- `macOS Apple Silicon`: primary tested build for this release
- `Windows x64`: packaged and published as `Alpha`; not hardware-tested before release

## Compatibility

Quail Ultra keeps the original Quail qbank folder format and reads older progress files with backward-compatible normalization.

## Release Channel

This is a `Late Beta` prerelease, not a `1.0` stable release.
